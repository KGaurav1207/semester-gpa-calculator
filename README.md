# GPA Calculator (Java)

A desktop Java Swing application to calculate semester GPA easily for NIT Mizoram CSE students.

## Description

This application provides a user-friendly interface to calculate GPA for each semester based on the subjects and their respective credits. It supports all 8 semesters of the CSE curriculum at NIT Mizoram.

## Features

- Semester-wise GPA calculation
- Simple and user-friendly UI using Java Swing
- Grade-to-point conversion system (AA=10, AB=9, BB=8, BC=7, CC=6, CD=5, DD=4)
- Pre-configured subjects and credits for each semester
- Real-time GPA calculation

## Prerequisites

- Java Development Kit (JDK) 8 or higher
- Java Runtime Environment (JRE) for running the application

## How to Run

1. Ensure you have Java installed on your system.
2. Compile the Java file:
   ```bash
   javac GpaCalculator.java
   ```
3. Run the application:
   ```bash
   java GpaCalculator
   ```

## Usage

1. Launch the application.
2. Click on the desired semester button (Semester 1 to Semester 8).
3. In the new window, enter the grades for each subject (e.g., AA, AB, BB, etc.).
4. Click "Calculate GPA" to see your semester GPA.

## Grade System

- AA: 10 points
- AB: 9 points
- BB: 8 points
- BC: 7 points
- CC: 6 points
- CD: 5 points
- DD: 4 points
- Other grades: 0 points
